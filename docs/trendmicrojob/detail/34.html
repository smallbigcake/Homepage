  • 工作职责： 
    This job is to work with R&D in the Software Development Life Cycle (SDLC) to assist the projects to release products with the advanced knowledge of Software Configuration Management(SCM) and Software Release Processes.

This position will work closely with RD and QA throughout SDLC. Like,

1. Management of release artifacts (source code, document and release build) with version control system (Perforce, Git).
2. Watching the new industrial software release technologies (version control, CI, CD, and etc.)
3. Plan, design, implement and optimize the CI/CD process with product development team.
4. Plan, design, implement and optimize the system environment which related to above area.
5. Develop and maintain new or existing tools that is used by release processes.
6. Provide essential software release info to product development team.
  • 任职资格：

Qualifications

1.Familiar with multiple DevOps relevant tools like: GitHub, Perforce, Jenkins, Jira, Docker, Kubernetes, Ansible, and etc.
2.Familiar with project management and agile practices, like Scrum.
3.Familiar with Cloud solution like: AWS, Azure, VMware
4.Strong communication skills and willing to take challenges.
5.Good English read/write and oral skill is required.
6.Experience of new system/service evaluation, auto-deployment and operation which is using by product development group.
7.Fast learning and has enthusiasm on doing problem analysis and problem solving.
8.3-year experience with scripting languages writing on Python, Perl and Shell script is essential
9.3-year experience with various OS management on Windows, Unix/Linux is essential.
10.Familiar with Java/C++ is a plus.
